## REACT FORM

REACT에서 form을 다루기 위한 연습 프로젝트입니다.

---

#### 요구조건
- 이메일/pw 받을 수 있는 인풋폼 2개
- 커스텀훅으로 로그인폼을 밸리데이션 할수 있는 훅
- 서브밋 버튼(모든 벨리데이션 에러가 없어야 서브밋 버튼 활성화)
- 이메일폼 : 이메일 형식이여야함
- Pw : 8자 이상. 최소한 하나 이상의 특수문자. 영대소문자 필요
- 각 벨리데이션 에러마다 적절한 에러메세지
- 만족되지 않았을 경우 적절한 에러 

#### 사용스택
- react
